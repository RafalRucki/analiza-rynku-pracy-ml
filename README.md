# Python Salary Analysis (NoFluffJobs)

## Project Goal
Analiza wynagrodzeń w ofertach pracy dla programistów Python w Polsce
na podstawie ofert z jawnymi widełkami płacowymi.

## Data Source
- NoFluffJobs REST API
- Oferty zawierające frazę "python"
- Waluta: PLN (monthly)

## Key Insights
- Wynagrodzenia rosną wraz z poziomem doświadczenia.
- Segment Senior charakteryzuje się dużym rozrzutem widełek płacowych.
- Oferty z jawnymi widełkami dotyczą głównie poziomów Mid i Senior.
- Rynek Juniorów jest marginalnie reprezentowany.

## Limitations
- Analiza obejmuje wyłącznie oferty z jawnymi widełkami płacowymi.
- Dominacja ofert zdalnych.
- Brak informacji o technologii, kontrakcie i formie zatrudnienia.
